# ls command

## What it does
Lists files or directories in the current or specifies directory

## Basic syntax
ls [options] [directory]

## Common options / examples
- ls             # List files in current directory
- ls -l          # Long format listing (permissions, owner, size)
- ls -a          # Show hidden files (starting with .)
- ls -la         # Combine long listing and show hidden files
- ls /etc        # List files in /etc directory
