# rm -r command

## What it does
Deletes directories and their contents recursively.

## Basic syntax
rm -r directory_name

## Common options / examples
- rm -r test             # Delete directory and contents
- rm -r dir1 dir2        # Delete multiple directories
- rm -r *                # Delete everything in current directory

## Notes / mistakes
- Very dangerous if used incorrectly
- rm -r / can destroy the system (never run)
