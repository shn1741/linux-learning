# touch command

## What it does
Creates empty files or updates file timestamps.

## Basic syntax
touch [options] file_name

## Common options / examples
- touch file.txt        # Create empty file
- touch a.txt b.txt     # Create multiple files
- touch -t 202601231200 file.txt  # Set specific timestamp

## Notes / mistakes
- Does not overwrite file contents
- Useful for quickly creating placeholder files
